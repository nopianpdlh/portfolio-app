## 🎨 Color Coding

### Certificate Status Badges:

```
┌───────────────────────────────────────────────────────────┐
│                                                           │
│  [Active ✓]        → Green badge, green shield icon      │
│  Certificate valid and current                            │
│                                                           │
│  [Expiring Soon ⚠] → Yellow badge, yellow alert icon     │
│  Certificate expires within 3 months                      │
│                                                           │
│  [Expired ✗]       → Red badge, red X icon               │
│  Certificate has expired                                  │
│                                                           │
│  No expiry date    → Blue text "No expiration"           │
│  Certificate never expires                                │
│                                                           │
└───────────────────────────────────────────────────────────┘
```

### Education Status:

```
┌───────────────────────────────────────────────────────────┐
│                                                           │
│  [Currently Studying] → Green badge                      │
│  End date shows "Present" in green                        │
│                                                           │
│  Completed          → No badge                           │
│  Shows actual end date                                    │
│                                                           │
└───────────────────────────────────────────────────────────┘
```

## ✨ Animation Sequence

### On Scroll:

```
1. Section Title fades in ↓
   "🎓 Certifications"
   
2. Description fades in ↓
   "Professional certifications..."
   
3. Cards stagger in (left to right) →
   Card 1 (delay: 0ms)
   Card 2 (delay: 100ms)
   Card 3 (delay: 200ms)
   
4. Stats fade in ↓
   "3 Total | 2 Active | 1 No Expiration"
```

### On Hover:

```
Certificate Card:
- Scale: 1.0 → 1.02
- Shadow: md → xl
- Border: transparent → primary/50
- Duration: 300ms

Education Card:
- Scale: 1.0 → 1.01
- Shadow: md → xl
- Border: transparent → primary/50
- Timeline dot: scale 1.0 → 1.25

Verify Button:
- Background: transparent → primary
- Text: muted → primary-foreground
```
